# [MU-JAVA]

Java helpers to handle RDF/Sparql & mu stack.

## Usage

```xml
  <repositories>
    <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
    </repository>
  </repositories>

  <dependency>
      <groupId>com.github.lblod</groupId>
      <artifactId>mu-java</artifactId>
      <version>${mu-java.version}</version>
  </dependency>
```
